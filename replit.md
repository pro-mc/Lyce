# Lyce

## Overview
A Node.js web application running on Express.

## Project Structure
- `index.js` - Main Express server entry point
- `package.json` - Node.js dependencies and scripts

## Running the Application
The app runs on port 5000 using Express. Start with:
```
node index.js
```

## Technical Stack
- Node.js 20
- Express.js
